Faking It

Welcome to Faking It! A fun multiplayer word game.

How to Play

Faking It is a game where players try to expose the Faker — the only person who didn’t receive the real question.

Setup

  • 3–8 players (best with 4–6)
  • One device per player or shared screen
  • 3 rounds, 3 questions per round
  • Each question has a new Faker

Round 1: Random Topics

Everyone answers the same question — except the Faker gets a twist.

  • Real Players: Answer the real question
    Example: “Who makes the best music?” → “Taylor Swift”
  • Faker: Answers a different (related) question
    Example: “Name a musician who's never won a Grammy.” → “Nicki Minaj”

Round 2: About the Group

Real players answer a question about someone in the room — the Faker must pretend they know what the question was.

  • Real Players: “Who here has the most money?” → “Sarah!”
  • Faker: Answers a different question with a name
    “Who here would need help paying rent?” → “Probably Alex”

Round 3: Numbers

Everyone gives a number — but the Faker’s question is unrelated.

  • Real Players: “How many times have you cried in the last month?” → “3”
  • Faker: “How many texts have you gotten today?” → “47”

Voting & Scoring

  • 30-second discussion
  • Everyone votes for the Faker
  • If Faker is caught: Real players +1 each
  • If Faker survives: Faker +2
  • After 3 rounds → highest score wins

Have fun, lie convincingly, and may the best bluffer win!
